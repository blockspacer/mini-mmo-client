# mini-mmo-client
The client for a very simple open world (not massively) multiplayer online game

## Requirements
* Protobuf compiler
* Protobuf library
* SFML 2.5.1
* TGUI 0.8.3
* CMake 3.1+

## Usage
To build:

```
$ mkdir build && cd build
$ cmake ..
$ make
```

To run:

```
$ ./build/bin/client
```
